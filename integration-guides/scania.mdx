To connect your Scania telematics data with PathStack, you will need the following access credentials:
* Client ID
* Secret Key

<Note>PathStack's Scania integration is currentlt in closed beta. For early access please contact us on 1300 822 370 or at support@nuonic.com.au</Note>

To obtain these credentials, follow the steps below:

<Steps>
  <Step title="Create a Scania developer user">
    Register a new user at the Scania [developer portal](https://developer.scania.com/signup). Make sure you select account type 'FMS developer' and that you have a Scania Data Access service enabled.
    <img src="/images/integration-guides/scania-1.png" />

    After logging in to the developer portal, click the 'Obtain keys' button on the Overview page. 
    <img src="/images/integration-guides/scania-2.png" />
  </Step>
  <Step title="Create a client and access the API credentials">
    On the next page, click the 'Create client' button and complete the form. The API access credentials will be generated and can then be copied from the page.
    <img src="/images/integration-guides/scania-3.png" />
  </Step>
</Steps>

<Tip>For futher assistance configuring your Scania API access, please contact the Scania team at support.developer.cs@scania.com</Tip>
