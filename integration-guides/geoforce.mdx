<Note>
  Geoforce offers Geotab devices, and proprietary devices. If you have GeoTab devices, connect with the Geotab
  integration, not the Geoforce integration. You can find the guide [here](/integration-guides/geotab). If you have
  Geoforce devices, please follow the guide below.
</Note>

To connect your Geoforce telematics data with PathStack, you will need the following access credentials:

- GraphQL API Key

Please see the guide for how to create the API Key [here](https://developers.geoforce.com/coding-examples/#:~:text=found%20here.-,Authentication,-To%20access%20the).
