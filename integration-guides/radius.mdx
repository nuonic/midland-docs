title: Radius Telematics
---

To connect your Radius telematics device integration with PathStack, you will need to be provided with an API Key and a Client Id for accessing the API. 

To find these credentials, please follow the steps below
<Steps>

<Step title = "Request  API details">

As a Radius customer, you will need to contact your account manager or the integrations team and request access to the API Key. You will need to be given an API key for accessing third party integrations.

https://www.radiustelematics.com/us/integrations/

</Step>
</Steps>

<Warning>For additional support and information, contact Radius Telematics at 0800 056 4432 or info@radiustelematics.com</Warning>
