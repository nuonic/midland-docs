title: Netstar Australia
---

To connect your Netstar telematics device integration with PathStack, you will need to be provided with an API URL
Prefix a Username and Password for accessing the API. The subsequent procedures will guide you through the process.

It is important to note that Netstar have two separate telematics platforms. If you are on the Ezy2c Fleet Manager, you
will not be be able to connect your telematics integration with PathStack as there is no available API.

You will need to contact your Netstar account representative and ask for the following information:

- API URL Prefix
- Username
- Password

You can also email customersupport@netstarus.com.au.

<Warning>
  For additional support and information, contact Netstar at 1800 325 052 or customer support@netstaraus.com.au.
</Warning>
