To connect your Volvo telematics device integration with PathStack, you will need to be provided with  a Username and Password for accessing the API. 

 
To find these credentials, please follow the steps below

<Steps>
  <Step title="Log into the Volvo Connect Account">
  Visit Volvo Login Bridgestone Webfleet service and log in to your account. Once logged in, navigate to the 'Users' section.
  <img src="https://365scotney-my.sharepoint.com/:i:/r/personal/sameer_nanduri_inloop_com_au/Documents/Pathstack%20Images/Volvo_1.png?csf=1&web=1&e=W7ABsh" />
  </Step>
  
  <Step title="Create an application">
  In order to generate your API key, please follow this link to create an application on Volvo connect and you will receive a VCC API Key.
  https://developer.volvocars.com/account/
  </Step>

  <Step title="Retrieve your credentials">
  After configuring the 'Volvo Connect' interface, your API Key will be provided. Ensure to note down or copy your API Key, as it will be required to integrate your Volvo telematics device with PathStack.
  </Step>

</Steps>

Require assiatnce on signing up or creating an application? Visit 
https://developer.volvocars.com/apis/docs/getting-started/
