To connect your WebFleet telematics device integration with PathStack, you will need to be provided with your Account name, a Username, password and API Key for accessing the API. 


To find these credentials, please follow the steps below

<Steps>

  <Step title = "Login to your Webfleet account">
Visit Webfleet Login Bridgestone Webfleet service and log in to your account. Once logged in, navigate to the 'Users' section.
  </Step>

  <Step title="Create a new user for PathStack">
   Within the 'Users' section, create a new user account with the email address set as 'Support@nuonic.com.au'. Ensure that the user account has the necessary permissions and access rights.

  <img src="https://365scotney-my.sharepoint.com/:i:/r/personal/sameer_nanduri_inloop_com_au/Documents/Pathstack%20Images/WF_1.png?csf=1&web=1&e=U1oX8M" />

  </Step>
<Step title="Configure Webfleet Connect interface">
   In the 'Interfaces' section of your Webfleet account settings, locate and select 'Webfleet Connect'. Confirming these changes will generate an API Key associated with your account.

  </Step>
  <Step title="Retrieve your credentials">
  After configuring the 'Webfleet Connect' interface, your API Key will be provided. Ensure to note down or copy your API Key, as it will be required to integrate your Bridgestone Webfleet telematics device with PathStack.
  <img src="https://365scotney-my.sharepoint.com/:i:/r/personal/sameer_nanduri_inloop_com_au/Documents/Pathstack%20Images/WF_2.png?csf=1&web=1&e=U1oX8M" />

See the link below Webfleet guide on how to retrieve your API Key
https://www.webfleet.com/en_gb/webfleet/partners/integration/developer-resources/#apikey

</Step>
</Steps>
