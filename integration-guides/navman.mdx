To connect your Navman telematics device integration with PathStack, you will need to be provided with an API Key for
accessing the API.

To find these credentials, please follow the process below:

As a Teletrac Navman customer, you will need to contact your account manager or the integrations team and request access
to the Open API. You will need to be given an API key for accessing third party integrations.

Visit Navman support by clicking the link below:

[Navman customer support](https://www.teletracnavman.co.uk/support/customer-resource/api-request)

<Warning>For additional support and information, contact Teletrac Navman at 1300 111 477. </Warning>
